输入框 ​

普通输入框 ​

js
<Linput  placeholder="请输入内容"></Linput>

禁用 ​

disabled

js
<Linput disabled placeholder="请输入内容"></Linput>

尺寸 ​

medium,small,mini,不传则默认

medium



small


mini
js
<Linput size="medium" placeholder="请输入内容"></Linput>
<Linput size="small" placeholder="请输入内容"></Linput>
<Linput size="mini" placeholder="请输入内容"></Linput>

一键清空 ​

当输入框有值时可一键清空

js
<Linput size="medium" v-model="tel" clearable placeholder="请输入内容"></Linput>

密码框 ​

js
<Linput size="medium" v-model="tel" type="password" showPassword placeholder="请输入密码"></Linput>

带icon输入框 ​



js
<Linput v-model="tel" clearable prefixIcon="bianji" placeholder="请输入内容"></Linput>
<Linput v-model="tel" suffixIcon="bianji" placeholder="请输入内容"></Linput>

复合输入框 ​

http://
js
<Linput v-model="tel" placeholder="请输入内容">
    <template #prepend>
        http://
    </template>
</Linput>


.com
js
<Linput placeholder="请输入内容">
    <template #append>
        .com
    </template>
</Linput>

自适应高度文本域 ​

设置autosize="{ minRows: 2, maxRows: 5 }最小行数为2，最大行数为5

js
<Linput type="textarea" :autosize="{ minRows: 2, maxRows: 5 }" v-model="text"></Linput>
